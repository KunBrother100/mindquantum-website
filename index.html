简单示例

from mindquantum import *

# 创建量子电路
circuit = Circuit()
circuit += H.on(0)
circuit += X.on(1, 0)

# 模拟电路
simulator = Simulator('projectq', 2)
simulator.apply_circuit(circuit)
print(simulator.get_qs())

系统要求

  • Python 3.7 或更高版本
